Welcome to the File Upload API

This API allows clients to upload files to the server.

How to Use

1️⃣ Start the server on Computer A:

python server.py --host 192.168.X.X --port 5001

2️⃣ Run the client on Computer B:

python client.py --server_ip 192.168.X.X --server_port 5001

3️⃣ Select a file and click Upload to send it to the server.

Ensure that both computers are connected to the same network to avoid connection issues.
