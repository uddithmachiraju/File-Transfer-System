Welcome to the File Upload API

This API allows clients to upload files to the server.

Use this route as a endpoint to send a file via a POST request.

How to Use

1️⃣ Make sure that server.py is running on Computer A.

2️⃣ Make sure that client.py is running on Computer B.

3️⃣ To upload a file from Computer B to Computer A, run the following command on Computer B:

python -m app.client <file_path>

Also, ensure that both computers have the same IP address configuration to make the upload successful.
